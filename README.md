# CASP Anonymous Review Code Package

This package contains the code and fixed numeric artifacts for the anonymous
review of the CASP dormant-return recommendation experiments.

## Contents

- `scripts/train_agknet/`: protocol builders, support audits, CASP solvers,
  learned gates, and reranker baselines.
- `src/`: shared baseline, data, model, and evaluation utilities.
- `figures/`: scripts used to regenerate the paper-facing numeric figures from
  fixed result artifacts.
- `results/`: small JSON/CSV/Markdown summaries for the reported tables.
- `docs/`: artifact ledger and reproduction notes.
- `tools/check_results.py`: lightweight consistency checks for reported numbers.

Raw datasets, generated protocol pickles, checkpoints, LaTeX sources, and paper
PDFs are intentionally not included. The datasets are public challenge datasets
or third-party logs and should be obtained from their original sources.

## Quick Smoke Test

From this directory:

```bash
bash scripts/run_smoke_tests.sh
```

The smoke test checks Python syntax, verifies the fixed result files used by the
main tables, and scans for common local-machine or archive artifacts. It does
not rerun the full experiments because the raw data and protocol cache files are
not part of the review package.

## Reproduction With External Data

Install the core dependencies:

```bash
python3 -m pip install -r requirements.txt
```

Place the datasets under `data/` following `data/README.md`, then use the
commands in `docs/REPRODUCTION.md`. Some commands expect intermediate protocol
artifacts generated by earlier steps; `docs/ARTIFACTS.md` maps each reported row
to the corresponding script family and fixed result summary.

## Review Scope

The included fixed result summaries are sufficient to audit the reported table
values and to check that the corresponding code paths are present. Full
end-to-end regeneration requires the external datasets and can take substantially
longer than the smoke test.
